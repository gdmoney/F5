Notes
=====
Changing the BIG-IP host name does not update the device name when the device is a member of a Device Service Clustering (DSC) device group.



Device Name
===========
Device Management > Devices > DEVICE > Change Device Name



Hostname
========
System > Platform > General Properties > Host Name
